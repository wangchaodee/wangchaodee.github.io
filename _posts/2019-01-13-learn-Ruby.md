## install rvm 
1. [site] <https://rvm.io/>
2. curl -sSL https://get.rvm.io | bash -s stable
3. use command to update rvm :rvm get stable  

## install ruby 
1. check avaliable ruby versions , $: rvm list known 
2. rvm install ruby 2.6
3. rvm use 2.6 --default
